# Components Catalog

React + TypeScript + Vite で構築されたUIコンポーネントカタログです。

## 技術スタック

- React 19
- TypeScript
- Vite
- Tailwind CSS
- Storybook 10

## セットアップ

```bash
npm install
```

## 開発

```bash
# 開発サーバー起動
npm run dev

# Storybook起動
npm run storybook
```

## ビルド

```bash
# アプリケーションビルド
npm run build

# Storybookビルド
npm run build-storybook
```

## コード品質

```bash
# Lint
npm run lint

# テスト
npx vitest --project=storybook
```

## コンポーネント

- **ToggleButton** - オン/オフ状態を切り替えるトグルボタン
