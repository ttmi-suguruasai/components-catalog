# Components Catalog

React + TypeScript + Vite で構築されたUIコンポーネントカタログです。

## 技術スタック

- React 19
- TypeScript
- Vite
- Storybook

## セットアップ

```bash
npm install
```

## 開発

```bash
# 開発サーバー起動
npm run dev

# Storybook起動
npm run storybook
```

## ビルド

```bash
# アプリケーションビルド
npm run build

# Storybookビルド
npm run build-storybook
```

## コンポーネント

- **ToggleButton** - オン/オフ状態を切り替えるトグルボタン
